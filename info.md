1. Предмет работ

1.1. Исполнитель выполняет для Заказчика работы по разработке и внедрению сервиса аналитики рекламных
кампаний, включающего:
• Telegram-бот для выгрузки и отображения статистики;
• сценарии автоматизации (воркфлоу);
· интеграции с API Яндекс Директ, VK Ads и Яндекс.Метрика;
• систему хранения и историзации данных в Google Sheets и/или базе данных;
веб-дашборд для визуальной аналитики.

1.2. Результатом работ является полностью работоспособный сервис отчётности, установленный и настроенный
на инфраструктуре Заказчика, с передачей исходных кодов, документации и доступов.
